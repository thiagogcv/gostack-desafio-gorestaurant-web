# Como rodar? 🤔
Siga os passos:

```
    yarn
    npm i -g json-server
    json-server server.json -p 3333
    yarn start

```
Happy Hacking! 🚀

⚠️ Caso o projeto não mostre os produtos, rode <code>adb reverse tcp:3333 tcp:3333</code> antes de iniciar a Fake API ⚠️

